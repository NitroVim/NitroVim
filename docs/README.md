# NitroVim Documentation

Welcome to the documentation for NitroVim, a powerful and premium Neovim configuration.

## Introduction

NitroVim is designed for performance, aesthetics, and productivity. It comes with a curated set of plugins, custom commands, and an intuitive keymap system.

- [Features](features.md): In-depth look at advanced features like AI, DAP, and UI components.
- [Customization](customization.md): Guide on how to customize and extend NitroVim.

## Quick Start

1.  **Leader Key**: The leader key is set to `<Space>`.
2.  **File Explorer**: Press `<leader>e` to toggle the file explorer.
3.  **Find Files**: Use Telescope to search for files (defaults to `<leader>ff` if not overridden, check keymaps).
4.  **Terminal**: Toggle the floating terminal with `<c-\>`.
