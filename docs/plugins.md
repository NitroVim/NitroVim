# Plugins

NitroVim is powered by a robust set of plugins managed by `lazy.nvim`.

## Core Infrastructure

- **[lazy.nvim](https://github.com/folke/lazy.nvim)**: Lightning-fast plugin manager.
- **[mason.nvim](https://github.com/williamboman/mason.nvim)**: Portable package manager for LSP servers, DAP servers, linters, and formatters.
- **[nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)**: Quickstart configurations for the Nvim LSP client.

## User Interface

- **[nvim-tree.lua](https://github.com/nvim-tree/nvim-tree.lua)**: A file explorer tree for neovim.
- **[bufferline.nvim](https://github.com/akinsho/bufferline.nvim)**: A snappy bufferline.
- **[lualine.nvim](https://github.com/nvim-lualine/lualine.nvim)**: A blazing fast and easy to configure Neovim statusline.
- **[dashboard-nvim](https://github.com/nvimdev/dashboard-nvim)**: A fancy start screen.
- **[noice.nvim](https://github.com/folke/noice.nvim)**: Highly experimental UI for messages, cmdline and the popupmenu.

## Productivity & Tools

- **[telescope.nvim](https://github.com/nvim-telescope/telescope.nvim)**: Highly extendable fuzzy finder over lists.
- **[toggleterm.nvim](https://github.com/akinsho/toggleterm.nvim)**: Multiple terminals, floating terminals, and more.
- **[auto-save.nvim](https://github.com/pocco81/auto-save.nvim)**: Automatically save your changes.
- **[gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim)**: Super fast git decorations.
- **[trouble.nvim](https://github.com/folke/trouble.nvim)**: A pretty diagnostics, references, telescope results, quickfix and location list.

## Editing Experience

- **[nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)**: Nvim Treesitter configurations and abstraction layer.
- **[nvim-cmp](https://github.com/hrsh7th/nvim-cmp)**: A completion engine plugin for neovim.
- **[nvim-autopairs](https://github.com/windwp/nvim-autopairs)**: Autopairs for neovim written in lua.
- **[nvim-surround](https://github.com/kylechui/nvim-surround)**: Add/change/delete surrounding delimiter pairs.
